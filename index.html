$(BlingJS)

A JavaScript library

by Jesse Dailey

BlingJS is named after the "$" operator, the default alias for the constructor.

$("body").append('<div></div>');

It's more than just a jQuery stand-in.

  • Double some item's widths: 
    $("pre").zap('style.width', -> @ * 2)
  • Generate 10 random strings: 
    $.zeros(10).map $.random.string 16
