BlingJS

A jQuery-like JavaScript framework.

(only for WebKit)

BlingJS is named after the bling operator ($) to which it is bound by default. ex. 

$("body").append('<div>');

Get the latest BlingJS. (minified).

Documentation

A "Bling" is an ordered set of items of any type. This set defines functional operations to collect and manipulate them.

You create Blings by wrapping up other items: 

$("button") // css expressions
$("<button>") // or html
$([1, "foo", {a: 'b'}]) // or junk

Then you can use functional primitives like: each, map, reduce, filter, distinct, skip, take, etc. to manipulate this list. For a full list, see the BlingJS Documentation

During all of the functional operations, the context (the 'this' value) is set to the item being iterated. 

$("button").each(function(){
	alert(this.nodeName);
})

This would alert once for every button on the page. Notice that the 'this' value is the raw element, not a Bling.

My favorites are zip and zap. The .zip(prop) method will extract a list of the values of a certain property. This is why .parent() is defined as simply as .zip('parentNode'), meaning, return a list of all the .parentNode properties of the items in the current set.

The following are all equivalent: 

$("button").parent().first()
$("button").zip('parentNode').first()
$("button").first().parentNode

The .zip() method is different from .attr(x), which in Bling is defined as .zip('getAttribute').call(x), and applies only to the Node attributes, not general object properties.

The .zap() method is similar to .zip() except that it sets the value of the property, returning the original set of nodes. 

// hide all buttons on the page
$("button").zap("style.display","none")

For the full API, consult the BlingJS Documentation

jQuery

BlingJS is very much like jQuery, and you can do just about anything in it that you can do in jQuery. If you know jQuery, then in general you can use BlingJS by writing what you would expect to work in jQuery, then assume everything you get is a list, and end your expressions with things like .first()
